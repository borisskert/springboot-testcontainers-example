# Why this repository

This solution is a little showcase for me to get familiar with [TestContainers](https://testcontainers.com/).

# Build and run test

## Build

```shell script
$ ./mvnw clean compile
```

## Run tests

```shell script
$ ./mvnw clean test
```

## Build package

```shell script
$ ./mvnw clean package
```
